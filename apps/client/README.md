# Logging Platform Client

Next.js dashboard frontend for the logging platform.

## Environment

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:5000
```

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

The Express API runs separately from `apps/server`.
